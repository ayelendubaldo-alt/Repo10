
# Instrucciones básicas de Git

## git init
Inicializa un repositorio nuevo en la carpeta actual.  
Ejemplo: git init

## git status
Muestra los cambios pendientes (archivos modificados, nuevos, etc.).  
Ejemplo: git status

## git add
Añade archivos al área de preparación (staging).  
Ejemplo: git add instruccions.md

## git commit
Guarda los cambios preparados en el repositorio local con un mensaje.  
Ejemplo: git commit -m "Añadidas instrucciones básicas de Git"

## git push
Envía tus commits locales al repositorio remoto.  
Ejemplo: git push origin instruccions-git
